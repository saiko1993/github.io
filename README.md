🔐 Privacy Policy

Last Updated: March 2026

1. Introduction

LiquidGlassVault (“the App”) is designed to securely store personal information such as passwords, API tokens, and sensitive data directly on your device.

Your privacy and data security are our highest priority.

⸻

2. Information We Collect

The App does not collect, store, transmit, or share any personal data with external servers.

All information entered into the App:
	•	Passwords
	•	API keys
	•	Tokens
	•	Card details
	•	Notes

is stored locally on your device only.

⸻

3. Data Storage & Security

All stored information:
	•	Remains on the user’s device
	•	Is not transmitted to any external server
	•	Is not shared with third parties
	•	Is not used for analytics or tracking

The App may use:
	•	Apple’s built-in encryption mechanisms
	•	Secure local storage frameworks
	•	Biometric authentication (Face ID / Touch ID)

to protect your data.

⸻

4. No Data Sharing

We do not:
	•	Sell data
	•	Share data
	•	Track users
	•	Use third-party advertising
	•	Use analytics SDKs

⸻

5. Internet Access

The App does not require internet access for core functionality.

If future updates introduce optional cloud sync features, users will be informed explicitly.

⸻

6. User Control

Users may:
	•	Add data
	•	Edit data
	•	Delete data at any time

Deleting the App will permanently remove all stored data unless backup features are enabled by the user.

⸻

7. Children’s Privacy

The App is not directed to children under 13 and does not knowingly collect any personal information.

⸻

8. Changes to This Policy
